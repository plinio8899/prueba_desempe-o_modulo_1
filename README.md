# Performance Test

Student management system, based on CRUD, using a JSON file as a database

## 📂 Project Structure

The project is divided into two main files:

* `main.py`: This is where you'll find all the logic of the program.
* `data.py`: Here you will find the dictionary used as a schema for the database and the CRUD functions.
* `json_config.py`: Here is everything related to saving and reading the JSON file used as a database.
* `db.json`: It's the database.

## 🛠️ Requirements

* **Python 3.x** or higher.


1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/plinio8899/prueba_desempe-o_modulo_1.git](https://github.com/plinio8899/prueba_desempe-o_modulo_1.git)
    cd prueba_desempe-o_modulo_1
    ```

2.  **Run the application:**
    ```bash
    python main.py
    ```
